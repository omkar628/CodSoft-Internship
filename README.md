#  CodSoft Machine Learning Internship

This repository contains my solutions for the **CodSoft Machine Learning Internship** tasks. Each project demonstrates the application of machine learning algorithms to solve real-world classification problems using Python and Scikit-learn.

##  Projects

###  Task 1 – Movie Genre Classification

* Predicts the genre of a movie from its plot summary.
* Uses **TF-IDF** for feature extraction.
* Models:

  * Logistic Regression
  * Random Forest
* **Best Accuracy:** **57.78%**

---

###  Task 2 – Credit Card Fraud Detection

* Detects fraudulent credit card transactions.
* Includes data preprocessing, feature engineering, label encoding, feature scaling, and **SMOTE** for handling class imbalance.
* Models:

  * Logistic Regression
  * Decision Tree
  * Random Forest
* **Best Model:** Random Forest
* **Accuracy:** **99.84%**
* **F1 Score:** **85.92%**

---

###  Task 3 – SMS Spam Detection

* Classifies SMS messages as **Spam** or **Ham**.
* Uses **TF-IDF** for text vectorization.
* Models:

  * Logistic Regression
  * Naive Bayes
  * Support Vector Machine (Linear SVC)
* **Best Accuracy:** **97.49%**
